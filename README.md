# Breakout
### About
Breakout is a simple 2D video game, originally published by Atari in 1976.

### Compilation
```Bash
make
```

### Clean Build
```Bash
make clean
```

### Execution
```Bash
./breakout
```

### Known Issues
- X11 library has memory leaks.
